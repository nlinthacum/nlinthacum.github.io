Wine Fridge Text Alerts

Project Description

Github repo

This project was made to get notified when the temperature inside my parent's wine fridge went outside of the desired range. They had been having issues with the wine freezing for no apparent reason. I used a thermocouple on a Raspberry Pi to receive sensor input and Twilio API to send message alerts. If the temperature goes outside of the range, an alert message is sent through Twilio. I also thought it would be cool to have the temperature sent to me when I wanted it, so I made a Python program that would send the temp when a text is sent to my Twilio number. The server I used to connect with Twilio from the pi was not 100% reliable, so I edited the boot file of the raspberry pi to run all of the necessary code, allowing the pi to be restarted and functioning by only toggling the power. Lastly, I added a smart switch to the wine fridge, so that I can remotely control the power to the fridge when I receive a text that the fridge is freezing the wine.
