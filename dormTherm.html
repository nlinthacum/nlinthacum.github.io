Dorm Room Thermostat

Project Description

Github repo: overt repo

This project started out as just a thermistor and an infrared LED hooked up to an Arduino with hopes of controlling my fan according to the temperature. I spent a lot of time trying to duplicate the IR signal that my fan's remote sends to turn it on and off which was made particularly hard since it doesn't follow a standard protocol. During this process, I came across an oLED display and my scope for this project immediately expanded: I could now make a user friendly display to control the temperature. Programming the display brought on a whole new complexity to the project, but made it much more useful and exciting. I added 2 buttons to control the display that allow the user to change the desired temp up and down, scroll through the mode options, and select the desired mode. The scope for the project was expanded once again when I spent several hours debugging code only to find that one of the ground wires had come loose on the breadboard. This painful discovery pushed me to design and fabricate a printed circuit board. I learned Autodesk's EAGLE to design the board. The board design worked great, except for the temperature sensor which had a different pin configuration than my component, resulting in a manual wiring of the pins. Overall, this project allowed me to expand my knowledge of I2C, PCB design, IR communication, and many elements of project organization.
